# line-slideshow-backend
LINEから画像を受け取り、スライドショーを表示するツールのバックエンド。
画像を受け取り、S3にアップロードするまでを実施。

作成はここを参照：https://zenn.dev/kou_pg_0131/articles/line-push-text-message
